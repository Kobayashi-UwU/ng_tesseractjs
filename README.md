## Project Detail
This project tests the performance of TesseractJS under different conditions (On MacOS, SamsungA05) and how long it takes to run TesseractJS.

## Test Result
```
GalaxyA05
Minimum Requirement
Ram : 2-4 GB
CPU : 1.5 GHz (Octa-core)
```
Model | GalaxyA05 Time 
--- | --- 
eng | 2.4 Sec 
tha | 2.0 Sec 
eng+the | 3.7 Sec
